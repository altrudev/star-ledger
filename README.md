# Star Ledger Integrated Codebase No Assets v2.1.6

This is an overwrite-ready integrated codebase.

The brightness fix is already appended into every stylesheet in `src/styles/`, and common root stylesheet names are included as fallbacks:

- `style.css`
- `styles.css`
- `main.css`
- `game.css`
- `index.css`

No separate wiring should be required if your hosted game already loads any stylesheet from this package.

Assets are not included.
