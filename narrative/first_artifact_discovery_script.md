# Close Proximity Ping — Canonical First Artifact Discovery

## Trigger
During travel, before the artifact system is unlocked.

## System Alert
PROXIMITY ALERT:
Unregistered mass detected.
Signal decay suggests long-duration drift.
Object is broadcasting a degraded cargo-lock ping.

## Player Choices
- Avoid Object
- Inspect Signal

## If Avoided
The ship adjusts vector and leaves the signal behind. For a long while, the proximity alert remains cached on the console like an unanswered question.

## If Inspected
The ping leads to scattered cargo ribs, frozen container plates, broken beacon housings, and one intact strongbox still clamped to a shattered freight spine.

The object is a **Grav-Lock Coffer**.

## Captain Lock-Crack
Most captains would need a bonded locksmith, a Navy warrant, or explosives.

You need twenty minutes, three bypass needles, a thermal patience drill, and the kind of education nobody lists on a clean résumé.

## Crew Aftermath
The crew gathers in the cargo bay without being called.

Nobody touches the object.

That is the first sign everyone understands what it might be.

Crew rumor:
“I heard stories about things like this. Old alien pieces. Some light up. Some open doors. Some make ships disappear from clean routes.”

Skeptic:
“And some do nothing?”

Answer:
“Some do nothing until someone finds the rest of them.”
